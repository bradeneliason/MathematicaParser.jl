# Notes on Mathematica Syntax

## Bracketing  
 - `(term)`	parentheses for grouping
 - `f[x]`    square brackets for functions
 - `{a,b,c}`	curly braces for lists
 - `v[[i]]`  double brackets for indexing ( Part[v,i] )
